# Password

Change your password in **Settings → Security**, under **Change password**.

1. **Enter your current password.** Type the password you use now.
2. **Choose a new password.** A live checklist confirms it meets each requirement: **10–128
   characters**, with an **uppercase letter**, a **lowercase letter**, a **number**, and a
   **special character**.
3. **Confirm and save.** Retype the new password, then select **Change password**. The button
   reads **Changing…** while it saves, and you'll see **"Password changed successfully."**

::: info

**Sign in with Google**. See [Create your account](/manual/getting-started/create-account).
:::
